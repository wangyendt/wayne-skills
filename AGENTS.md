# AGENTS.md

本文件为其他大模型/代理在本仓库协作时的统一入口说明。

## Single Source Of Truth

- 规范主文档：`CLAUDE.md`
- 当 `AGENTS.md` 与 `CLAUDE.md` 有冲突时，以 `CLAUDE.md` 为准。

## 必须遵循

1. Skill 命名、目录结构、清理规则：严格按 `CLAUDE.md` 执行。
2. README 编写与更新：严格按 `CLAUDE.md` 中“README 编写规范（经验总结）”执行。
3. 所有文档改动应与仓库当前真实内容一致（尤其是 `**/SKILL.md` 清单与数量）。

## 建议工作流

1. 先读 `CLAUDE.md` 再动手改文件。
2. 修改后自检路径、命名、数量是否一致。
3. 提交信息清晰描述变更范围（如 `docs: ...` / `feat(skill): ...`）。
